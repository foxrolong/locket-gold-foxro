## <h2 align="center">🚀 Hướng dẫn sử dụng</h2>
### Cách 1: Tải trực tiếp
1. **Truy cập repo** [repo]().
2. **Tải về** Danh sách Module.

### Cách 2: Thêm bằng URL
1. **Sao chép URL**

```Locket
https://raw.githubusercontent.com/foxrolong/locket-gold-foxro/refs/heads/main/locket.foxrolong
```
```Locket - youtube
https://raw.githubusercontent.com/foxrolong/locket-gold-foxro/refs/heads/main/foxro-youtube-locket.conf
```
2. **Mở Shadowrocket** → Vào **Configuration > Module**.  
3. **Nhấn `+`** (góc phải) → **Dán URL** → Nhấn `OK`.  
4. **Bật VPN**
